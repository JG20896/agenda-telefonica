# Agenda Telefónica PHP

Una agenda telefónica web desarrollada en PHP con interfaz responsive.

## Características
- Lista de contactos
- Búsqueda en tiempo real
- Agregar nuevos contactos
- Diseño responsive

## Instalación
1. Clonar repositorio
2. Configurar base de datos (si aplica)
3. Acceder via navegador

## Tecnologías
- PHP
- HTML5
- CSS3
- JavaScript
